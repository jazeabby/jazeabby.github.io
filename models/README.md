# 3D Models (optional)

Place space-related GLB/GLTF files here to replace the procedural planet and moon.

- **planet.glb** – Replaces the central blue planet (position 0,0,0; scale 1.5).
- **station.glb** – Replaces the small orbiting moon (position 4,0,1; scale 0.8).

**Sources (free, space-themed):**

- [Kenney Space Kit](https://kenney.nl/assets/space-kit) – CC0, GLTF/GLB.
- [Kenney Modular Space Kit](https://kenney.nl/assets/modular-space-kit) – CC0.
- [Sketchfab](https://sketchfab.com) – Search "planet" or "space station", filter Downloadable + Free; download GLB; add attribution if required.
- [Poly Pizza](https://poly.pizza) – Astronaut/space models, often CC Attribution.

If these files are missing, the scene uses procedural spheres as fallback.
